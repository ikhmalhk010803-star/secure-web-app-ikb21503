# Secure Web Application – IKB21503

## Project Description
This project is a secure web application developed for the IKB21503 course. 
The system is designed and implemented following secure software development practices based on OWASP Top 10 and OWASP ASVS.

## Features
- User registration and login
- Role-Based Access Control (Admin and User)
- Secure CRUD module
- User profile management
- Audit logging for security-related events

## Security Implementation
- Password hashing using industry-standard algorithms
- Input validation on all user inputs
- Protection against SQL Injection using ORM / prepared statements
- Cross-Site Scripting (XSS) prevention through output encoding
- Cross-Site Request Forgery (CSRF) protection
- Secure session management
- Secure configuration using environment variables

## Security Testing
- Manual code review checklist
- Static analysis tools
- Dynamic security testing using OWASP ZAP
